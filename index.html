Layout de Largura Fixa

Este layout possui uma largura fixa de 800 pixels e não se ajusta ao tamanho da janela do navegador.

Layout Líquido

Este layout usa uma largura de 80% da janela do navegador e se ajusta ao tamanho da janela.

Layout Elástico

Este layout usa uma largura de 50em e se ajusta ao tamanho da fonte e ao tamanho da janela do navegador.

Layout Híbrido

Este layout combina largura líquida e largura fixa. Ele se ajusta ao tamanho da janela do navegador, mas não é menor que 300px ou maior que 1000px.
